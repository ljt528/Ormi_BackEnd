응답과 요청이 있고 그 사이에는 브라우저가 껴있음
★HTTP에는 Header와 Body가 있다.

클라이언트와 서버 사이에는 응답과 요청이 있다.

GET : 데이터 조회, URL 포함
POST : 데이터 생성, URL 불포함. Body 포함
PUT : 데이터 수정, URL 불포함. Body 포함
DELETE : 데이터 삭제

action : 주소
method : get, post 등...

URL구조
프로토콜 + 도메인 + 포트 + 경로 + 쿼리

요청을 보낼 때 꼭 주소가 필요하다!

★HTTP 요청 시 : URL + 메소드
★HTTP 응답 시 : 응답 코드 + 응답 Body
